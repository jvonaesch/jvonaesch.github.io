Jupyter Setup

Ihr könnt Jupyter Notebook auf der Try Jupyter-Seite kostenlos ausprobieren. Es ist allerdings empfehlenswert, Notebook direkt auf Eurem System zu installieren und lokal zo nutzen. Dazu installiert Ihr zuerst Jupyter mit pip, dem Python-Paketmanager:

pip install jupyterlab
pip install notebook

Nun könnt Ihr Notebook mit folgendem Befehl ausführen:

jupyter notebook
